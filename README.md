# Number Guessing Game

## Description
A simple Python program that generates a random number and challenges the user to guess it. The program compares the user's guess with the generated number and displays the result.

## Features
- Random number generation
- User input handling
- Guess validation
- Result display

## Technologies Used
- Python

## How to Run
1. Clone the repository.
2. Open the project folder.
3. Run the program using:

```bash
python Number_guessing.py
```
